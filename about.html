About This Website

Elite Car Showcase is a website created to present a collection of stylish, powerful, and well-known vehicles. The purpose of this site is to highlight different types of cars and their design.

This project was built using HTML and CSS with a clean and simple layout. The gallery page displays six selected cars with short descriptions for each one.
